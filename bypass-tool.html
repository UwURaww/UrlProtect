Setup Guide

Learn how to get your keys and configure each provider.

⚠️ The B.Y.P.A.S.S c-api-key is different from your provider key. You need both.
1

Join the B.Y.P.A.S.S Discord

Go to discord.gg/tsnNVWTkp5 and join the server. This is where you register and get support.

2

Register for API access

Follow the instructions in the Discord server to register your account and request an API key. Look for a #register or #get-key channel.

3

Copy your c-api-key

Once approved, you'll receive a key that looks like a long string of letters and numbers. Copy it exactly.

4

Paste it here

Click the ⚙ API Key button in the top-right corner of this page, paste your key, and hit Save. It's stored only in your browser.

✓ Linkvertise Dynamic (lvdynamic) is recommended — it uses your numeric User ID, which is easiest to find.
1

Go to Linkvertise

Visit linkvertise.com and log in or create a free account.

2

Find your User ID (for lvdynamic)

After logging in, go to your Dashboard. Look at the URL — it will look like linkvertise.com/user/dashboard/1229349. The number at the end is your User ID. Use that as your pikey.

3

Get a Bearer Token (for linkvertise provider)

If you use the linkvertise provider instead, go to Dashboard → API Settings. Generate or copy your Bearer Token and use that as your pikey.

⚠️ Don't mix them up — lvdynamic uses the numeric UserID, linkvertise uses the Bearer Token.
1

Go to LootLabs

Visit lootlabs.gg and log in or sign up.

2

Open the Developers section

In your dashboard, navigate to Developers or API settings. Look for an API Key field.

3

Generate and copy your API Key

Generate a new key if you don't have one, then copy it and paste it as your pikey.

4

Configure Postback (Required)

LootLabs requires postback to work with B.Y.P.A.S.S. Set your postback URL in the LootLabs dashboard. Check the B.Y.P.A.S.S Discord for the exact postback URL to use.

5

Max Tasks

LootLabs supports a "Max Tasks" setting (1–5). When you select LootLabs as your provider, this option will appear on the form. Higher = more revenue but more work for the user.

1

Go to Work.ink

Visit work.ink and log in or create an account.

2

Open API settings

Go to your dashboard and find the API section. Copy your API Key.

3

Paste as pikey

Use your Work.ink API Key as the pikey value in the form.

These providers all follow the same pattern — log in, find your API Key in the dashboard settings, and use it as your pikey.

✓ pikey = "Provider Key" — it's the credential that lets B.Y.P.A.S.S create links on your behalf with your monetization account.

Each provider uses a different type of credential. Here's a quick reference:

Provider pikey Type Where to Get It
lvdynamic numeric User ID Dashboard URL
linkvertise Bearer Token Dashboard → API Settings
lootlabs API Key Dashboard → Developers
workink API Key Dashboard → API
cutyio API Key Settings → API
lockr API Key Dashboard → API
rinku API Key Dashboard → API
⚠️ Keep your pikey private — it's tied to your monetization account. Anyone with it can create links on your behalf.
